- Backend build: `cd backend; dotnet build SenicBilling.slnx`
- Frontend build: `cd frontend; npm run build`
- Frontend lint: `cd frontend; npm run lint`
- Frontend dev server: `cd frontend; npm run dev`

## High-Risk Modules

- `DocumentNumberGeneratorService`: uniqueness and concurrency.
- `SenicBillingDbContext`: tenant relationships, decimal precision, indexes, seed data, migrations.
- `AuthController` / `JwtTokenService`: token claims, secret handling, tenant and role claims.
- `PaymentsController` / `OmisePaymentService`: payment state, webhook integrity, idempotency.
- `InvoiceForm` / document calculations: VAT mode, discount, grand total, frontend/backend parity.
- `A4DocumentTemplate` / print CSS: Thai document layout, A4 accuracy, mobile/print differences.
