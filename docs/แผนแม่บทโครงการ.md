# **แผนแม่บทโครงการ (Project Master Plan)**

**โครงการ:** Senic Billing Next

**ระยะเวลาดำเนินการโดยประมาณ:** 4-6 เดือน

## **1\. วัตถุประสงค์โครงการ**

1. พัฒนาระบบออกเอกสารการเงิน 4 โมดูลหลัก (ใบเสร็จ, บิลเงินสด, ใบส่งของ, ใบกำกับภาษี) บนสถาปัตยกรรม Cloud  
2. ออกแบบ UX/UI ใหม่ทั้งหมด ภายใต้คอนเซปต์ "Professional, Futuristic, yet Warm"  
3. รองรับการทำงานแบบ Cross-platform ผ่าน PWA

## **2\. ขอบเขตการทำงาน (Scope of Work)**

* **Phase 1: Discovery & Design (เดือนที่ 1\)**  
  * รวบรวม Requirement เชิงลึก  
  * ออกแบบ Wireframe & UI/UX Design System (รองรับ Dynamic Themes)  
  * ออกแบบ Database Schema (PostgreSQL)  
* **Phase 2: Core Development \- Backend & API (เดือนที่ 2-3)**  
  * พัฒนาระบบ API ด้วย C\# .NET 10 และ Entity Framework Core  
  * วางระบบ Authentication & Authorization (JWT)  
  * ตั้งค่าฐานข้อมูล PostgreSQL และ CI/CD Pipeline  
* **Phase 3: Core Development \- Frontend & PWA (เดือนที่ 3-4)**  
  * พัฒนาหน้าจอด้วย React 19 \+ Vite \+ TypeScript  
  * เชื่อมต่อ API (Integration) และทำ State Management  
  * พัฒนาระบบ Dynamic Theming (Orange-Brown / Ocean Blue)  
* **Phase 4: Analytics & PDF Generation (เดือนที่ 5\)**  
  * พัฒนาระบบ Dashboard เชิงลึก  
  * พัฒนาระบบ Export PDF/Excel ที่มีความแม่นยำสูง  
* **Phase 5: UAT, QA & Deployment (เดือนที่ 6\)**  
  * System Testing & User Acceptance Testing  
  * Security Auditing  
  * Production Deployment on Cloud (e.g., Azure / AWS)

## **3\. การบริหารความเสี่ยง (Risk Management)**

* **ความเสี่ยงด้านการเปลี่ยนผ่านเทคโนโลยี:** คัดเลือกนักพัฒนาที่มีความเชี่ยวชาญเฉพาะด้าน .NET 10 และ React 19  
* **ความเสี่ยงด้านข้อมูลลูกค้าเดิม:** เตรียมสคริปต์ Migration ข้อมูลจากระบบเดิม (Legacy Database) สู่ PostgreSQL อย่างรัดกุม
